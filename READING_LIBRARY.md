```javascript
{
    meaning: "牌的含义和关键词",

    future: "未来位置的解读文案",
```

| **advice** | 针对该牌的建议或启示 | "相信直觉，勇敢迈出第一步。" |


## 📝 编写文案的建议

### 风格指南

1. **保持一致的语气**
   - 使用第二人称（"你"）地址读者
   - 保持神秘而富有启发性的语气
   - 避免过于绝对或悲观的措辞

2. **结构清晰**
   - 每部分 1-2 句话
   - 避免过长或复杂的句子
   - 使用平易近人的语言

3. **内容相关**
   - past/present/future 应反映该位置的含义
   - meaning 应包含 3-5 个关键词
   - advice 应是实用的建议或启示

### 示例分析

以"愚者"为例：

```javascript
'0. The Fool (愚者)': {
    // meaning: 核心含义（短而精）
    meaning: '新的开始、冒险精神、无限可能',
    
    // past: 叙述背景，这张牌对过去意味着什么
    past: '过去：你充满了探索欲望，敢于尝试新事物。
            那时的你无所畏惧，充满了对未来的憧憬。',
    
    // present: 提供指导，这张牌对现在的建议
    present: '现在：是时候相信你的直觉了。
             眼前虽有未知，但正是这份未知给予你无限的可能性。',
    
    // future: 描绘可能性，这张牌预示的未来
    future: '未来：新的旅程即将展开。
            无论是新的关系、工作还是冒险，都将为你的人生注入新的活力。',
    
    // advice: 核心启示（精炼实用）
    advice: '相信直觉，勇敢迈出第一步。风险中隐藏着机遇。'
}
```

```

   ↓
2. 选择"普通解牌"

3. 根据牌名查询 READING_LIBRARY

```

```
[对应位置的解读文案]
```


## 🔧 技术细节

### getDefaultReading() 函数

```javascript
function getDefaultReading(cardInfo) {
    // 查询文案库
    const library = READING_LIBRARY[cardInfo.name];
    
    if (!library) {
        return `【${cardInfo.name}】\n\n文案库中暂无该牌的完整解读，请等待补充。`;
    }

    // 根据牌的位置选择解读
    const position = currentReadingIndex;
    let reading = '';

    if (position === 0) {
        reading = `【${cardInfo.name}】\n\n${library.past}`;
    } else if (position === 1) {
        reading = `【${cardInfo.name}】\n\n${library.present}`;
    } else if (position === 2) {
        reading = `【${cardInfo.name}】\n\n${library.future}`;
    }

    // 添加关键词和建议
    reading += `\n\n关键词：${library.meaning}\n建议：${library.advice}`;
    return reading;
}
```

## 💡 使用提示

### 编辑时注意

1. **保持牌名一致**
   - 确保字段名称与 TAROT_DATA 中的牌名完全相同
   - 例：`'0. The Fool (愚者)'`

2. **格式正确**
   - 使用英文引号 `'` 作为字符串边界
   - 使用英文逗号 `,` 分隔字段
   - 注意 JSON 格式的有效性

3. **避免错误**
   - 最后一个字段后不要加逗号
   - 特殊字符要转义（如 `\"` 表示引号）
   - 定期保存并测试

### 验证修改

修改后，可以：

1. **打开浏览器控制台** (F12 → Console)
2. **输入验证命令**:
   ```javascript
   console.log(READING_LIBRARY['0. The Fool (愚者)']);
   ```
3. **检查输出**是否包含你的修改
