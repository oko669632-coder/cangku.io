## 🔑 API 密钥快速获取

### DeepSeek (推荐，便宜)
1. https://platform.deepseek.com
2. 注册 → API Keys → Create
3. URL: `https://api.deepseek.com/v1`
4. Model: `deepseek-chat`

### OpenAI (功能强大)
1. https://platform.openai.com/account/api-keys
2. Create new → Copy
3. URL: `https://api.openai.com/v1`
4. Model: `gpt-3.5-turbo` (便宜) 或 `gpt-4` (强大)
