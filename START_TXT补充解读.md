## 💡 快速开始流程

### 步骤 1：打开 txt 文件
```
数据库补充22张卡牌解读.txt
↓
用文本编辑器打开（推荐 VS Code）
```

### 步骤 2：编辑补充内容
```
找到要编辑的卡牌
【卡牌】0. The Fool (愚者)
【补充解读】
[修改这里的内容]
【来源】[修改来源]
```

### 步骤 3：保存并刷新
```
1. 保存文件（Ctrl+S）
   - 编码必须是 UTF-8
   
2. 打开/刷新塔罗应用
   - 点击"进入仪式"
   - 选择"普通解读"
   
3. 查看效果
   - 补充内容会在原有解读下方显示
   - 带 📚 符号标记
```

## 🔑 关键要点

### ✅ 必须做的：
1. **卡牌名称完全匹配** - 包括大小写和符号
2. **编码为 UTF-8** - 否则中文会乱码
3. **保持格式** - 【卡牌】【补充解读】【来源】缺一不可
4. **修改后保存** - Ctrl+S 保存文件

### ❌ 不要做的：
1. **改变卡牌名称** - 否则无法匹配
2. **删除格式标记** - 【卡牌】等标记必须保留
3. **使用 GB2312 编码** - 必须是 UTF-8
4. **同时编辑多个副本** - 会导致冲突
