# 祭坛主题图片资源

本文件夹用于存放祭坛主题的本地图片资源。

## 图片文件列表

请确保以下图片文件存在：

1. `celtic_altar.png` - 凯尔特风格祭坛图片
2. `egyptian_altar.png` - 埃及风格祭坛图片
3. `cosmic_altar.png` - 宇宙风格祭坛图片
4. `forest_altar.png` - 森林风格祭坛图片

## 获取图片的方法

### 方法1：使用在线图片生成API

您可以使用以下URL生成图片，然后保存到本文件夹：

- 凯尔特风格：`https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=Celtic%20tarot%20altar%20with%20ancient%20symbols%20and%20green%20elements&image_size=square_hd`
- 埃及风格：`https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=Egyptian%20tarot%20altar%20with%20hieroglyphics%20and%20golden%20elements&image_size=square_hd`
- 宇宙风格：`https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=Cosmic%20tarot%20altar%20with%20stars%20and%20galaxies&image_size=square_hd`
- 森林风格：`https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=Forest%20tarot%20altar%20with%20nature%20elements%20and%20greenery&image_size=square_hd`

### 方法2：使用自定义图片

您也可以使用自己喜欢的图片，只要确保文件名与上述一致即可。

## 图片要求

- 建议尺寸：至少1024x1024像素
- 格式：PNG或JPG
- 风格：与对应的主题相符

## 注意事项

如果图片文件不存在，程序会尝试从在线API加载图片，但可能会较慢。建议使用本地图片以获得最佳性能。
