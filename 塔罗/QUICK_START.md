# 快速配置参考

## 🚀 快速开始（3 分钟）

### Step 1: 获取 API 密钥
选择一个 AI 服务商（推荐 DeepSeek 或 OpenAI）：
- **DeepSeek**: https://platform.deepseek.com (便宜，效果好)
- **OpenAI**: https://platform.openai.com (功能强大)

### Step 2: 启动应用
打开 `index.html` 并点击"进入仪式"

### Step 3: 配置 AI
右上角 ⚙️ 面板：
1. 打开"启用 AI 解牌"开关 ✓
2. 填入对应的 API 配置（见下表）
3. 点击"测试连接" ✓
4. 点击"保存设置" ✓

## ❓ 常见配置问题

### 问：测试连接失败？
答：检查以下项：
1. ✓ API 密钥是否复制完整（无多余空格）
2. ✓ API URL 是否正确（特别注意末尾有无 `/v1`）
3. ✓ 网络连接是否正常
4. ✓ API 账户余额是否充足

### 问：解牌很慢？
答：可能原因：
- 网络延迟
- 模型响应慢（试试切换模型）
- API 服务器负载高（稍候重试）

### 问：显示默认解读？
答：说明 AI 未启用或配置有误。检查：
- 开关是否打开 ✓
- 是否点击保存设置 ✓
- 配置是否完整（所有字段都有值）

### 问：想换模型？
答：在配置面板修改"模型"字段即可，例如：
- `gpt-4` → `gpt-3.5-turbo`
- `deepseek-chat` → `deepseek-coder`

## 🔑 API 密钥获取流程

### DeepSeek (最简单)
1. https://platform.deepseek.com/api_keys
2. 注册 → 登录 → 创建密钥
3. 复制密钥
4. API URL: `https://api.deepseek.com/v1`

### OpenAI
1. https://platform.openai.com/account/api-keys
2. 登录 → Create new secret key
3. 复制密钥
4. 绑定支付方式
5. API URL: `https://api.openai.com/v1`
