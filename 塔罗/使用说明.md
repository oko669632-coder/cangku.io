# 🔮 塔罗牌 AI 解牌 - 使用说明

## 快速开始 (3 步)

### 1️⃣ 启动应用
打开 `index.html` 文件

### 2️⃣ 进入仪式
点击按钮 "进入仪式"

### 3️⃣ 执行手势
- **握拳** 🤜 → 洗牌（会听到魔法音效）
- **张掌** 🖐️ → 铺开牌阵
- **伸食指** ☝️ ×3 → 依次抽取过去、现在、未来三张牌

---

## 配置 AI (可选)

### 第一步：获取 API 密钥

**推荐方案：DeepSeek (免费额度充足，便宜)**
1. 访问 https://platform.deepseek.com
2. 注册账户
3. 进入 "API Keys" 页面
4. 点击 "Create API Key"
5. 复制密钥

### 第二步：配置应用

1. 启动应用后，点击右上角 **⚙️** 图标
2. 打开 "启用 AI 解牌" 开关（变成金色 ✓）
3. 填写以下信息：

| 字段 | 值 |
|-----|-----|
| API 基础 URL | `https://api.deepseek.com/v1` |
| API 密钥 | 粘贴你的密钥 |
| 模型 | `deepseek-chat` |
| 解牌语言 | `中文` |
| 解牌风格 | `神秘学派` |

4. 点击 "测试连接" 验证配置 ✓
5. 点击 "保存设置"

## 完整占卜流程

```
1. 进入仪式 👉 "进入仪式" 按钮

2. 握拳洗牌 👉 面对镜头握紧拳头（能看到效果会更好）
              听到"嗡~嗡~嗡~"音效，卡牌在洗牌

3. 张掌铺开 👉 张开手掌（所有手指伸直）
              卡牌自动散开成扇形

4. 抽取第一张 👉 伸出食指，系统随机选择一张牌
              牌向上飞来，落在左侧位置
              下方显示"过去"和牌名

5. 抽取第二张 👉 再次伸出食指
              牌落在中间位置
              下方显示"现在"和牌名

6. 抽取第三张 👉 第三次伸出食指
              牌落在右侧位置
              下方显示"未来"和牌名

7. 牌牌翻转 👉 三张牌自动翻转显示正面 ✨

8. 进入解牌 👉 弹出解牌面板
              如果配置了 AI，会显示 AI 生成的解读
              如果没配置，显示默认解读

9. 浏览解读 👉 点击"下一张"查看下一张牌的解读
              或点击"完成仪式"结束

10. 新仪式  👉 重新握拳可以开始新的占卜
```

## 常见手势问题

### ❌ 握拳没有反应？
- 确保手完全握紧（五个手指都在拳头里）
- 摄像头能清楚看到手
- 手离摄像头距离适当（50-100cm）

### ❌ 张掌没有反应？
- 五个手指都要伸直张开
- 不要做出其他手势
- 手要在摄像头前方足够久（约 1 秒）

### ❌ 食指没有被识别？
- 确保只有食指是伸直的
- 其他手指都要弯曲
- 不要太快移动

### ✅ 调试技巧
- 试试增加光线（靠近光源）
- 检查摄像头是否清洁
- 刷新页面重试

## AI 解读失败的处理

### 显示 "✗ 灵能感应中断"？
这说明 API 调用失败，但别担心！应用会自动显示默认解读。

**检查清单：**
- [ ] API 密钥是否正确（复制时没有多余空格）
- [ ] API URL 是否正确（特别是末尾的 `/v1`）
- [ ] 网络连接是否正常
- [ ] API 服务是否正常运行

### 显示 "✨ 灵能解读（AI 未启用）"？
这说明 AI 功能未启用，显示的是默认解读。

**解决方案：**
1. 右上角 ⚙️ → 打开 "启用 AI 解牌" 开关
2. 填写 API 配置
3. 点击"保存设置"
4. 重新进行占卜

## 设置重置

### 清除所有配置
在浏览器开发者工具（F12）的 Console 标签粘贴：
```javascript
['ai_enabled', 'ai_api_url', 'ai_api_key', 'ai_model', 'reading_language', 'reading_style'].forEach(k => localStorage.removeItem(k));
```

然后刷新页面即可。

## 性能优化建议

### 第一次加载可能较慢？
- 应用需要加载卡牌图片，首次加载 5-10 秒属于正常
- 之后打开会从浏览器缓存加载，速度很快

### AI 解读响应慢？
- 某些模型响应较慢（如 GPT-4）
- 可以切换到更快的模型（如 `gpt-3.5-turbo`）
- 检查网络连接

### 想要离线使用？
- 保存网页为 HTML 文件
- 禁用 AI 功能使用默认解读
- 完全离线运行

## 常见问题

**Q: 没有 API 密钥也能用吗？**  
A: 可以！默认会显示静态解读，功能完整。

**Q: 可以自己训练模型吗？**  
A: 可以，但需要自己搭建私有 API 服务，然后配置 API URL。

**Q: 支持手机使用吗？**  
A: 可以，但需要允许浏览器访问摄像头。

**Q: 可以保存占卜结果吗？**  
A: 目前不支持，可以手动截图或记录。

**Q: 可以自定义卡牌吗？**  
A: 可以修改代码中的 `TAROT_DATA` 数组。
