# 🎯 快速参考指南

## 一句话总结
您可以编辑 **数据库补充22张卡牌解读.txt** 文件来为所有 22 张卡添加补充解读，应用会自动加载显示。

---

## 三个步骤使用

### ①编辑
打开 `数据库补充22张卡牌解读.txt` → 修改补充内容 → 保存（UTF-8）

### ②刷新
打开或刷新浏览器中的塔罗应用 → 点击"进入仪式"

### ③查看
进行卡牌抽取 → 选择"普通解读" → 看到补充内容（带📚符号）

## 关键提醒

🔴 **保存时必须选 UTF-8 编码！**

### VS Code 保存为 UTF-8
右下角找到编码 → 点击 → 选"UTF-8" → 保存

### Windows 记事本保存为 UTF-8
另存为 → 编码选"UTF-8" → 保存

## 常见问题速答

**Q: 改了没显示？**
A: ①文件保存了吗（Ctrl+S）②编码是 UTF-8 吗③浏览器刷新了吗（Ctrl+F5）

**Q: 显示乱码？**
A: 用 VS Code 打开 → 右下角选 UTF-8 → 保存

**Q: 找不到卡牌？**
A: 复制本表中的卡牌名称，粘贴到【卡牌】后面

**Q: 能删除某张卡吗？**
A: 可以，把补充内容留空即可

**Q: 能加表情吗？**
A: 可以！任何 UTF-8 字符都支持

## 出错了怎么办

### 现象：补充内容不显示

**检查顺序：**
1. 卡牌名称是否完全匹配？
2. 文件是否保存？
3. 浏览器是否刷新？
4. 浏览器控制台有无错误？(F12)

**解决：**
一个一个检查上面的项目

### 现象：显示乱码

**原因：** 编码不是 UTF-8

**解决：**
1. 用 VS Code 打开 txt
2. 右下角看编码（如 GBK）
3. 点击 → 选"UTF-8"
4. Ctrl+S 保存
